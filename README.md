# Filmógrafo

Website de filmes e séries desenvolvido para a unidade curricular Tecnologias Web e Ambientes Móveis.

## Framework

O website foi desenvolvido em Angular, com TypeScript para o backend e HTML/CSS para o frontend.

## APIs Utilizadas

- TMDB API: comunicação com a API para obtenção de dados;
- Supabase: criação de base de dados e API para implementação de um sistema de utilizadores.

## Estrutura do Repositório

- README.md para descrição do projeto
- gitignore para ignorar dados sensíveis e ficheiros desnecessários
- filmografo, diretoria onde fica o código

## Chaves de API

Adicionar as chaves das APIs num ficheiro com o caminho `src/environments/environment.development.ts` com a mesma estrutura do ficheiro `environment.ts`
